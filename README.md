# Simulador Funcional UFLA-RISC
Este diretório contém o simulador funcional em Python para o processador RISC de 32 bits, conforme especificado no arquivo uRISCUFLA2025_2.pdf.
## Arquivos
- `simulator.py`: O código principal do simulador, contendo a lógica de fetch, decode e execute para as 22 instruções.
- `test_program.py`: Um script de exemplo para demonstrar o uso do simulador com um programa de teste simples.
## Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. Navegue até o diretório `ufla_risc_simulator`.
3. Execute o programa de teste:
```bash
python3 test_program.py
```
